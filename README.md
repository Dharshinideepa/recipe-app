# 🍽️ Recipe App

Hey there! 👋 Welcome to my Recipe App!

This is one of my first projects as a 3rd year Engineering student

I was scared to start but here we are! 🎉

## 💡 What does this app do?
It's a simple app where you can store and view recipes.
Nothing fancy yet — but hey, every big project started small! 😄

## 🛠️ What I used to build this
- **Node.js** — to run the backend
- **Express.js** — to handle the routes
- **MongoDB** — to save the recipes
- **HTML/CSS** — for the frontend look

## 📁 How the files are organized
recipe-app/
├── public/      → the HTML pages you see
├── routes/      → handles all the app paths
├── models/      → the recipe data structure
└── server.js    → the heart of the app ❤️
